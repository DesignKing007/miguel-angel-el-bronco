What is a Set?

A set is an associated group of images or web pages that have the same set name. Sets are optional. You don't need to use them. If you don't want to associate a GreyBox link with another one, simply specify that this link is not part of a set.

Here are some examples of sets...

Images:

6 Figure List PowerEffects Million Dollar Roundtable

Web Pages:

Launch Google Launch Yahoo

Note that when you open any one of those links within the same set, you can page through them. Each set is separate and distinct from any other sets on yoru web page. What sets them apart is the set name you choose for each link. Those links that have the same set name are grouped together as in the examples above.
